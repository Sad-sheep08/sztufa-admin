# 深圳技术大学足球协会系统 - 管理后台 (sztufa-admin)

本项目是校园足球信息管理平台的**后台管理端**，供协会管理员管理球队、球员信息以及录入比赛日程和实时关键事件。

## 🛠️ 技术栈
* **核心框架**：React 18 + TypeScript
* **构建工具**：Webpack 5
* **图片直传**：集成 Cloudflare R2 对象存储，上传图片在前端动态转换为 `FormData` 流上传，替换 Base64 格式
* **API 通信**：基于 `axios` 封装，支持根据访问的域名（`localhost` 或生产域名）自适应动态切换请求地址

## 🌟 核心功能
1. **用户认证**：登录后自动缓存 JWT Token，并通过拦截器自动附加到请求头。
2. **球队管理**：
   * 创建、更新球队基本信息。
   * 支持通过 R2 接口直传高分辨率队徽。
3. **球员管理**：
   * 关联特定球队，进行球员信息的增删改查。
   * 支持球员照片直传及号码查重。
4. **赛事事件实时管理**：
   * 管理比赛日程、地点、比分。
   * **比赛关键事件流录入**：支持普通进球 ⚽、点球 🎯、乌龙球 🥅、换人 🔄、黄牌 🟨、红牌 🟥。
   * **智能业务规则强校验**：
     * 换人事件：联动“换上球员”和“换下球员”双向列表，校验防重，已被换下的球员不可二次换上。
     * 得分联动：主队得分 = 主队（普通进球 + 点球）+ 客队乌龙球。自动校验得分一致性，防止误录。

## 🚀 快速启动

### 1. 安装依赖
```bash
npm install
```

### 2. 本地开发调试
```bash
npm start
```
开发服务器默认在 `http://localhost:3000` 启动，并会自动代理 `/api` 请求至本地后端。

### 3. 生产构建
```bash
npm run build
```
编译产物会输出至 `build/` 目录。
